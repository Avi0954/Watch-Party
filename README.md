# Watch Party 🎬

A full-stack real-time synchronized YouTube player.

## Setup

### Backend
1. `cd apps/api`
2. `pip install -r requirements.txt`
3. `python main.py`

### Frontend
1. `cd apps/web`
2. `npm install`
3. `npm run dev`

### Docker
`docker-compose up --build`
